6. Insert images into your Word documents

### Option 2: View in GitHub/GitLab
- Simply open `DIAGRAMS_MERMAID.md` in GitHub or GitLab

### Option 3: VS Code with Extension

2. Open `DIAGRAMS_MERMAID.md`
